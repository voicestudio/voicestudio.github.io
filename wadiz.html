자주 묻는 질문

음원 저작권은 어떻게 되나요?
사용자가 직접 제작하거나 라이선스를 확보한 음원만 사용해야 합니다. Suno AI, Udio 등 AI 음원 생성 도구로 만든 100% 오리지널 음원이면 저작권 문제 없이 수익화 가능합니다.
Gemini API 키는 어떻게 받나요?
Google AI Studio(aistudio.google.com)에서 무료로 발급 가능합니다. 일일 무료 쿼터 내에서는 비용이 발생하지 않습니다.
AI가 만든 영상도 수익화가 되나요?
네. AI 생성 음원 + AI 생성 이미지로 만든 영상은 저작권 문제 없이 YouTube 수익화가 가능합니다.
다채널 운영 시 업로드 횟수는?
계정(아이디) 기준입니다. Ultimate로 6개 채널 운영 시 하루 총 6개까지 업로드 가능. 채널당 1개씩 배분하면 6채널에 매일 자동 업로드됩니다.
Super Ultimate는 뭐가 다른가요?
Ultimate 라이선스 3개를 제공합니다. 아이디 3개에 각각 적용하면 1대 PC에서 앱 1개로 최대 18개 채널 완전자동화가 가능합니다.
MacOS에서도 사용 가능한가요?
현재 Windows 정식 지원입니다. MacOS는 추후 지원 예정입니다.
환불은 어떻게 되나요?
와디즈 펀딩 환불 정책을 따릅니다. 라이선스 미활성 시 14일 이내 전액 환불 가능합니다.
