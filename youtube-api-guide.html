YouTube API 등록 가이드

본인 YouTube 채널 연결하기

아래 단계를 따라하면 약 7~10분 안에 끝납니다. 신용카드 등록 안 해도 됩니다.

▌ 1부. Google Cloud 가입 & 프로젝트 생성

1

Google Cloud 약관 동의

  1. console.cloud.google.com 접속
  2. 본인 Google 계정으로 로그인
  3. 첫 접속 시 약관 동의 다이얼로그 표시됨
  4. 국가: 대한민국 선택
  5. 서비스 약관 체크박스 2개 모두 체크
  6. 오른쪽 하단 "동의 및 계속하기" 클릭
⛔ 상단 "무료로 시작하기" 배너 누르지 마세요.
누르면 신용카드 등록이 요구됩니다. 그냥 "닫기" 또는 "X" 로 무시하면 됩니다.
2

프로젝트 선택 드롭다운 열기

  1. 약관 동의 후 Google Cloud Console 메인 화면 진입
  2. 화면 좌측 상단의 "프로젝트 선택" 드롭다운 클릭
  3. 새 창(팝업) 형태로 프로젝트 목록이 열림
3

"새 프로젝트" 버튼 클릭

  1. 프로젝트 선택 팝업의 오른쪽 상단 확인
  2. "새 프로젝트" 버튼 클릭
  3. 새 프로젝트 만들기 화면으로 이동
4

프로젝트 이름 입력 & 만들기

  1. 프로젝트 이름: 원하는 이름 입력 (예: my-cozy-uploader)
  2. 위치(조직)는 그대로 두기 (없음)
  3. 아래 "만들기" 버튼 클릭
  4. 몇 초 후 "프로젝트 생성됨" 알림이 뜸
💡 프로젝트 이름은 본인이 알아볼 수 있는 어떤 이름이든 OK. 영문/숫자/하이픈만 사용 가능.
5

방금 만든 프로젝트 선택

  1. 다시 좌측 상단 "프로젝트 선택" 드롭다운 클릭
  2. 방금 만든 프로젝트 이름 클릭
  3. 화면 상단의 프로젝트 이름이 변경됨 확인
⚠️ 이 단계를 빼먹으면 다음 작업이 다른 프로젝트에 적용됩니다. 반드시 방금 만든 프로젝트가 선택돼야 합니다.

▌ 2부. YouTube Data API v3 활성화

6

좌측 햄버거 메뉴 펼치기

  1. 화면 좌측 상단의 햄버거 메뉴(☰) 아이콘 클릭
  2. 좌측 사이드바 메뉴 펼침 표시
7

"API 및 서비스" → "라이브러리"

  1. 펼쳐진 메뉴에서 "API 및 서비스" 항목 찾기
  2. 하위 메뉴에서 "라이브러리" 클릭
  3. API 라이브러리 화면으로 이동
바로가기 링크
8

"YouTube Data API v3" 검색

  1. API 라이브러리 화면 상단 검색창 찾기
  2. 검색창에 YouTube Data API v3 입력
  3. 아래에 검색 결과로 YouTube Data API v3 카드가 표시됨
9

"YouTube Data API v3" 카드 클릭

  1. 검색 결과에서 빨간 YouTube 아이콘이 있는 "YouTube Data API v3" 카드 클릭
  2. API 상세 정보 화면으로 이동
10

"사용" 버튼 클릭 (활성화)

  1. API 상세 화면에서 파란 "사용" 버튼 클릭
  2. 활성화 처리 (몇 초 ~ 수십 초 소요)
  3. 처리 완료 후 자동으로 API 관리 화면으로 이동
11

YouTube Data API v3 활성화 완료 확인

  1. API/서비스 세부정보 화면 진입
  2. 화면 우측 "상태" 항목이 "사용 설정됨" 인지 확인
  3. 좌측 메뉴에 새 항목들이 표시됨
✅ 절반 완료! 이제 본인 인증 설정으로 넘어갑니다.

▌ 3부. 본인 인증 설정

12

OAuth 동의 화면 진입

  1. 왼쪽 사이드바 → "API 및 서비스"
  2. 하위 메뉴 "OAuth 동의 화면" 클릭
  3. "Google 인증 플랫폼이 아직 구성되지 않음" 안내 화면이 나오면 정상 진입
13

"시작하기" 클릭

  1. 안내 화면 중앙의 파란 "시작하기" 버튼 클릭
  2. 4단계 마법사가 열리면 정상
14

앱 정보 4단계 마법사 입력

  1. 앱 이름: My Cozy Uploader 입력
    • ⚠️ "Google" / "YouTube" 같은 단어 사용 금지
  2. 사용자 지원 이메일: 본인 이메일 선택
  3. 대상: "외부(External)" 선택
  4. 연락처 이메일: 본인 이메일 입력
  5. 약관 체크 → "만들기" 클릭
15

테스트 사용자 추가 (⭐ 필수)

이 단계를 빠뜨리면 본인 계정으로도 로그인 불가.

  1. 왼쪽 메뉴 대상 탭 클릭
  2. "테스트 사용자" 섹션의 "+ 사용자 추가" 클릭
  3. 본인 Google 이메일 입력 (YouTube 채널 소유 계정)
    • 여러 채널이면 각 채널 소유 이메일 모두 추가
16

테스트 사용자 저장

  1. "저장" 클릭
  2. 추가된 이메일이 목록에 표시되는지 확인
⛔ "앱 게시" 절대 금지.
"테스트 중" 상태 유지 → 본인/테스트 사용자만 쓸 수 있고 만료 없음. "게시" 누르면 Google 심사 필요.
17

클라이언트 탭으로 이동

  1. 왼쪽 메뉴 클라이언트 탭 클릭
  2. 상단 "+ 클라이언트 만들기" 버튼 클릭
18

"데스크톱 앱" 선택

  1. 애플리케이션 유형: "데스크톱 앱" 선택
    • ⚠️ "웹 애플리케이션" 아님 — 반드시 데스크톱
19

클라이언트 ID 만들기

  1. 이름: Cozy Uploader Desktop (원하는 이름 가능)
  2. "만들기" 클릭
20

JSON 다운로드

  1. 생성 완료 팝업에서 "JSON 다운로드" 클릭
  2. client_secret_xxxx.apps.googleusercontent.com.json 파일 저장
✅ 이 파일 절대 공유 금지. 본인 프로젝트의 인증 열쇠입니다. Cozy 외부에 노출되면 안 됩니다.
21

Cozy Uploader 에 JSON 등록

  1. Cozy Uploader 앱 실행 → 로그인
  2. 왼쪽 사이드바 → "설정" 탭
  3. 스크롤 → "YouTube API 인증 파일" 항목
  4. "파일 등록" 클릭 → 다운로드한 JSON 선택
  5. "등록 완료" 메시지 확인
22

"계정 연결" 클릭

  1. 설정 탭 → "YouTube 채널 → 계정 연결" 버튼 클릭
  2. 기본 브라우저가 자동으로 열림
23

Google 계정 선택

  1. 브라우저에서 본인 Google 계정 선택
  2. YouTube 채널 소유 계정으로 로그인
  3. 여러 계정이 있으면 채널 소유 계정만 선택
24

"확인되지 않은 앱" 경고 — 고급 클릭

OAuth 로그인 시 "Google에서 이 앱을 확인하지 않았습니다" 경고가 뜹니다. 본인 프로젝트이므로 안전합니다.

  1. 경고 화면 → "고급" 또는 "Advanced" 링크 클릭
경고가 뜨는 이유: 본인이 만든 앱을 Google 심사받지 않았기 때문. 본인 프로젝트에서 본인 계정 로그인은 안전합니다.
25

"(앱 이름)으로 이동" 클릭

  1. 고급 펼침 영역 → 하단 "(앱 이름)(안전하지 않음)(으)로 이동" 클릭
26

모든 권한 허용 (⭐ 중요)

  1. 권한 요청 화면 → 모든 체크박스 선택 ("모두 선택" 권장)
  2. 하단 "허용" 또는 "계속" 클릭
⚠️ 권한 누락 주의.
체크박스 하나라도 빠뜨리면 업로드/분석/예약 기능 중 일부가 작동 안 합니다. 반드시 전체 선택.
27

인증 완료

  1. "인증 완료! 탭을 닫으세요" 메시지 확인
  2. 브라우저 탭 닫기
  3. Cozy Uploader 앱으로 돌아가서 채널 이름이 표시되는지 확인
🎉 모든 설정 완료! 이제 Cozy Uploader 에서 본인 채널에 영상 업로드/예약/자동화가 가능합니다.

자주 묻는 질문

Q. 유료인가요?
무료입니다. 신용카드 등록 안 해도 개인 채널 운영에 충분합니다.
Q. 여러 YouTube 채널도 가능한가요?
가능합니다. 다른 채널의 Google 이메일도 15단계 "테스트 사용자" 에 추가하면 됩니다.
Q. 화면이 영어로만 나와요.
Google Cloud Console 우측 상단 프로필 → 설정 → 언어에서 "한국어" 로 변경 가능합니다. 영어 그대로 진행해도 단계는 동일합니다.
