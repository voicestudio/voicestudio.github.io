YouTube API 등록 가이드

본인 Google Cloud 프로젝트 연결하기

YouTube 업로드 쿼터(일일 10,000 units)를 본인 계정에 귀속시키기 위해 Google Cloud에서 client_secrets.json을 만들어 앱에 등록합니다. 약 5~7분 소요.

왜 이 작업이 필요한가요?
YouTube API 쿼터는 OAuth 클라이언트가 속한 Google Cloud 프로젝트 단위로 부과됩니다. 모든 사용자가 하나의 프로젝트를 공유하면 쿼터가 금방 소진되기 때문에, 각자 본인의 프로젝트를 만들어 연결해야 안정적으로 업로드할 수 있습니다.
1

Google Cloud Console 접속 & 프로젝트 생성

  1. console.cloud.google.com 에 접속 (본인 Google 계정으로 로그인)
  2. 상단 프로젝트 선택 드롭다운 클릭 → "새 프로젝트" 버튼 클릭
  3. 프로젝트 이름 입력 (예: my-cozy-uploader) → "만들기" 클릭
  4. 생성 완료 알림이 뜨면 상단 드롭다운에서 방금 만든 프로젝트 선택
💡 팁: 처음 Google Cloud를 사용하면 약관 동의 화면이 나올 수 있습니다. 무료 계정으로 시작하시면 됩니다.
2

YouTube Data API v3 활성화

  1. 왼쪽 메뉴 → "API 및 서비스" → "라이브러리"
  2. 검색창에 YouTube Data API v3 입력 후 선택
  3. "사용" 버튼 클릭 (활성화 완료)
직접 이동 링크
3

OAuth 동의 화면 구성

  1. 왼쪽 메뉴 → "API 및 서비스" → "OAuth 동의 화면"
  2. User Type: "외부" 선택 → "만들기"
  3. 앱 정보 입력:
    • 앱 이름: My Cozy Uploader (또는 원하는 이름)
    • 사용자 지원 이메일: 본인 Google 이메일 선택
    • 개발자 연락처 정보: 본인 Google 이메일 입력
  4. 나머지는 전부 기본값으로 두고 "저장 후 계속" 반복 클릭
  5. 대시보드로 돌아오면 "테스트 사용자" 섹션에서 "+ 사용자 추가" → 본인 Google 이메일 추가
⚠️ 중요: "앱 게시" 버튼은 누르지 마세요. 본인만 사용하는 앱이므로 "테스트 중" 상태로 두면 됩니다 (Google 심사 불필요).
4

OAuth 클라이언트 ID 생성

  1. 왼쪽 메뉴 → "API 및 서비스" → "사용자 인증 정보"
  2. 상단 "+ 사용자 인증 정보 만들기" → "OAuth 클라이언트 ID"
  3. 애플리케이션 유형: "데스크톱 앱" 선택
  4. 이름: Cozy Uploader Desktop (자유롭게)
  5. "만들기" 클릭
  6. 생성 완료 팝업에서 "JSON 다운로드" 클릭 → client_secret_xxxx.json 파일 저장
✅ 다운로드한 파일은 절대 공유하지 마세요. 이 파일이 곧 본인 프로젝트의 인증 열쇠입니다.
5

Cozy Uploader에 파일 등록

  1. Cozy Uploader 앱 실행 → 로그인
  2. 왼쪽 사이드바 → "설정" 탭
  3. "YouTube API 인증 파일" 항목 찾기
  4. "파일 등록" 버튼 클릭 → 다운로드한 client_secret_xxxx.json 선택
  5. "등록 완료" 메시지 확인
  6. 동일 설정 탭의 "YouTube 채널 > 계정 연결" 버튼 클릭
  7. 브라우저가 열리면 본인 Google 계정으로 로그인
6

"확인되지 않은 앱" 경고 통과

OAuth 로그인 시 "Google에서 이 앱을 확인하지 않았습니다" 경고 화면이 뜹니다. 본인이 만든 프로젝트이므로 안전합니다.

  1. 경고 화면에서 "고급" 또는 "Advanced" 링크 클릭
  2. 하단의 "[앱 이름](안전하지 않음)(으)로 이동" 클릭
  3. 권한 요청 화면에서 "허용" 클릭 (YouTube 업로드/관리/분석 권한)
  4. "인증 완료! 이 탭을 닫고 앱으로 돌아가세요" 메시지 확인 → 브라우저 탭 닫기
  5. Cozy Uploader로 돌아와서 채널 이름이 표시되는지 확인
경고가 뜨는 이유: 본인이 만든 앱을 Google에 심사받지 않았기 때문입니다 (심사받으려면 복잡한 절차 필요). 본인 프로젝트에서 본인 계정으로 로그인하는 경우 이 경고를 통과해도 안전합니다.

자주 묻는 질문

Q. 유료인가요?
Google Cloud 자체는 무료로 시작할 수 있고, YouTube Data API는 일일 10,000 units의 무료 쿼터를 제공합니다. 일반적으로 영상 1개 업로드당 약 1,700 units를 사용하므로 하루 5~6개까지 무료로 업로드 가능합니다.
Q. 하루 10개 이상 업로드하고 싶어요.
Google Cloud Console에서 쿼터 확장을 신청할 수 있습니다. Professional 등급(채널 6개 × 6영상 = 36개/일)을 사용하려면 확장 신청이 필요하며, 심사에 1~2개월 소요될 수 있습니다.
Q. 본인 확인 없이 사용하고 싶어요.
"OAuth 동의 화면"을 "테스트 중" 상태로 두고 본인을 테스트 사용자로 추가하면 별도 확인 없이 사용 가능합니다. "앱 게시"를 눌러야만 Google 심사가 필요해집니다.
Q. 여러 YouTube 채널을 하나의 프로젝트로 관리할 수 있나요?
네. client_secrets.json 하나로 여러 채널(같은 Google 계정 내)을 OAuth로 연결할 수 있습니다. Cozy Uploader 앱의 "채널 프로필" 기능을 사용하세요.
Q. 파일이 해킹당하면 어떻게 되나요?
Google Cloud Console에서 해당 OAuth 클라이언트를 삭제하고 새로 만들면 즉시 무효화됩니다. 또한 앱은 이 파일을 암호화된 Windows Credential Manager에 저장합니다.
🎉 등록이 완료되었다면, Cozy Uploader로 돌아가서 수동 업로드 또는 완전자동화 기능을 사용해보세요. 쿼터 부족으로 업로드가 실패할 때는 Google Cloud Console의 "API 및 서비스 > 쿼터"에서 현재 사용량을 확인할 수 있습니다.
