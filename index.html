AI가 매일 음악 영상을
자율적으로 만드는 도구

Cozy Uploader는 음악 플레이리스트 채널 크리에이터를 위한 데스크톱 앱입니다. AI가 배경 이미지, 썸네일, 메타데이터를 생성하고 FFmpeg로 렌더링 후 YouTube에 자동 예약 업로드합니다. Lofi / Jazz / Ambient 채널을 운영하는 모든 분들을 위한 자동화.

주요 기능

음악 파일 하나만 준비하면 나머지는 AI가 알아서.

🎨

AI 배경 이미지 생성

Google Imagen 기반 AI가 10개 스타일 프리셋으로 분위기에 맞는 배경을 생성합니다. 지브리, 사실적 사진, 로파이, 네온 시티 등.

🎬

FFmpeg 영상 렌더링

2K/1440p 해상도, 오디오 비주얼라이저, Ken Burns 효과, 인트로 자막까지 FFmpeg로 시네마틱 렌더링.

🤖

AI 메타데이터 자동 생성

제목·설명·태그를 Gemini가 CTR 최적화 관점에서 자동 작성. 타임스탬프 트랙리스트도 자동 삽입.

📅

자율 운영 모드

자정에 6개 영상을 계획→생성→예약 업로드, 새벽 3시에 성과 분석 → 학습. 시간이 지날수록 더 똑똑해집니다.

🌏

타임존 자동 대응

한국 거주자가 미국·유럽·일본 시청자 대상 채널 운영 시 타겟 타임존 기준으로 최적 시간대 자동 예약.

🔒

완전한 개인 계정 사용

YouTube 인증 정보는 PC에만 저장됩니다. 서버로 전송하지 않으며, 사용자 본인 채널에만 접근합니다.

자주 묻는 질문

YouTube 계정이 필요한가요?

네, 업로드할 대상 YouTube 채널의 Google 계정이 필요합니다. 앱은 OAuth 2.0으로 인증하며, 사용자 본인 채널에만 접근합니다. 이미 만들어진 영상의 메타데이터는 수정하지 않습니다.

Gemini API 키는 어떻게 발급받나요?

Google AI Studio(https://aistudio.google.com)에서 무료로 발급 가능합니다. 일일 무료 쿼터 내에서는 비용이 발생하지 않습니다.

AI가 생성한 이미지·영상의 저작권은 누구 것인가요?

Google Gemini API 이용약관에 따라 사용자(발급자)에게 있습니다. 단, 생성 결과물이 기존 저작물과 유사할 수 있으므로 YouTube 업로드 전 확인 권장.

업로드한 음원의 저작권은 어떻게 처리되나요?

사용자가 직접 제작하거나 정당하게 라이선스를 확보한 음원만 업로드해야 합니다. 저작권 침해 콘텐츠 업로드로 인한 책임은 사용자에게 있습니다.

다채널 운영 시 업로드 횟수는 어떻게 되나요?

Ultimate 등급은 최대 6개 채널을 연결할 수 있으며, 하루 총 6개 영상을 업로드할 수 있습니다. 채널당 1개씩 6채널에 올려도 되고, 한 채널에 6개를 몰아서 올려도 됩니다. 자율 모드에서 채널별 자동 배분도 가능합니다.

Super Ultimate는 뭐가 다른가요?

Super Ultimate는 Ultimate 라이선스를 3개 제공합니다. 서로 다른 아이디(계정) 3개에 각각 적용할 수 있어, 팀이나 파트너와 함께 여러 채널을 독립적으로 운영할 수 있습니다.

MacOS / Linux에서 사용 가능한가요?

현재 정식 지원은 Windows입니다. MacOS 빌드는 Pro+ 등급 한정 베타로 2026년 하반기 제공 예정입니다.

환불 정책은 어떻게 되나요?

와디즈 펀딩은 와디즈 환불 정책을, 공식 판매 채널은 이용약관을 따릅니다. 라이선스 미활성 시 14일 이내 전액 환불 가능합니다. 자세한 사항은 이용약관을 확인해주세요.
